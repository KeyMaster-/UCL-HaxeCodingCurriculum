Game development in Haxe - Project overview

Overview: The teaching material can be found in the 'webpages' folder, starting with index.html.
The final game that the material will be building up to can be found in the 'game/build' folder, and the associated code is located in 'game/src'.

Requirements: For this project, the student only has to download the haxe compiler (as instructed in Lesson 0), and has to have a modern browser installed.

Tasks Completed
  • Create the game framework.
  • Create the final game that the student will be creating.
  • Simplify the game code, and also make it more engaging.
  • Create the introduction page and overall page templates
  • Create lessons 0 to 7.
  • Create intermediate game code versions for each of the lessons, for the student to check against or a teacher to check solutions.
