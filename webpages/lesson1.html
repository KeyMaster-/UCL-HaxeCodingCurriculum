Lesson 1

Data Types, Variables and If Statements

The main function of any computer program is to collect, store and process data in one way or another. Therefore before learning anything else, we need to understand what Data Types and Variables are, as these are key to knowing how to process data.

Data Types


"Data Types" is a term used to refer to the different forms of data that exist in a program. For instance, numbers such as 5 or 500 are called Integers (or Int for short), whereas letters or a group of letters such as “f” or “five” are known as a String. We need to specify what type of data we are using in a program because these various types of data take up different amounts of memory, also, certain operations can only be performed on some of these types. For instance we cannot divide 5 by the String “hello”.

Here is a full list of the different data types available in Haxe:

  • String - A letter or group of letters. E.g. “f” or “five”

  • Integer - Numbers such as 5 or 500, or even hexadecimal numbers like 0xFA.

  • Float - Decimal point numbers such at 1.5 or 100.523. Scientific notation can be used here as well. e.g. 2e4 (2 x 10^4)

  • Boolean - A type that can only be one of two values, true or false.

  • Dynamic - This data type can be any type, and can even change types during the course of a program.

  • Unknown - A data type that we have not yet set. This gets set automatically once you give it some data.

  • Void - A way to represent no type, for example when there is no data.


    • Variables


    In any program, variables are the containers for the data that we use. These variables are given a name so that we can keep track of them easily. No two variables can have the same name. When we want to create a variable to contain our data, we “declare” the variable by giving it a name and the type of data that it will contain. This is known as “Declaration”.

    Example

    var shipName:String = “Galactica”;

    In this example we use the var keyword to indicate that we are going to declare a variable. shipName is the name we are giving to this variable, and String is the type of data that can be contained within it. We also initialise the variable by giving it some data straight away. In this case our ship is called the “Galactica”.
    If we wanted to rename our ship later in the program we can do this by simply writing the following line:

    shipName = “Colonial One”;

    Notice we do not need to use the keyword var or specify a type again as we have already declared the variable and given it name which we can use to refer to it later in the program.



    If Statements


    One of the most useful tools we have as programmers are known as If Statements.
    If statements are used when we only want lines of code to be executed if a condition is met. For example, we would only want the game over screen to appear if the player has reacher 0 hit points.
    Here is how that would look like in Haxe: 

    if (health <= 0){
        gameOver(); 
    }

    In this case we use the keyword if followed by a condition thats surrounded by round brackets ( ).
    We then put all the code we want to execute if the condition is true, after the condition surrounded by curly brackets { }.

    But what do we do if the condition is not true?
    This is where we use the else keyword. This will execute the code after the else keyword if the original if condition is not met.

    if (health <= 0){
        gameOver(); 
    }
    else{
        keepPlaying();
    }

    We can also have multiple if statements in a row in case we need to set multiple conditions. 
    if (health <= 0){
        gameOver(); 
    }
    else if (health < 10){
        flashScreen();
        keepPlaying();
    }

    In this case the program checks the first condition, if that is not met then it moves on to the next condition, and so on.

    Exercise 1


    Now lets put what we've learned to use in our game.

    To start, open the file Player.hx in your favourite text editor.
    Then go to line 31. This section of code is going to deal with our ships movement.

    What we want to do here is to allow our ship to move if we press certain keys. For instance, if we press the up arrow key, the ship should move upwards.
    In order to achieve this, we need two variables called move_x and move_y.
    These will store the integer values that make our ship move along the x or y axis of the screen.

    Then we need to determine if a player is pressing the keys we want them to press.
    To do this we use the function 

    Framework.input.keydown(n)

    Where n is a number representing a key.
    Do some googling to figure out which numbers correspond to the keys you want to use.

    Finally, use your knowledge of If Statements to make the ship move left, right, up and down if the player presses the right keys.

    Hint: You may need two sets of If Statements. One for left and right movements and the other for up and down movements.
