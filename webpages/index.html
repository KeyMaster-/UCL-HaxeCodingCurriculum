Game development in Haxe

Welcome to Coding Curriculum - Battlecode Galacticulum!

This project aims to teach basic coding to secondary school students (ages 13-17) through development of an arcade-style game. The game that students will develop is a side-scrolling spaceship game with score. The language that will be used for this project is Haxe (see Lesson 0 for setup details). No prior experience in coding is required for students to enjoy this project. Each lesson will consist of implementing or modifying a certain aspect of the game by adding to and changing the skeleton code provided in the downloadable files. At the end of the project, students should have successfully developed their own fully working and playable game!

Coding concepts that will be introduced include variables, data types, operators, control structures (if, while, for), functions, scope, and classes/objects.
